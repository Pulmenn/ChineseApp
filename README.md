Как запустить проект

Установите Expo Go на телефон (Android).

Убедитесь, что телефон и компьютер подключены к одной Wi-Fi сети.

В папке проекта откройте терминал и выполните:
npm install
npx expo start

Отсканируйте QR-код в открывшемся окне браузера с помощью камеры телефона или прямо в приложении Expo Go.
Приложение откроется на телефоне. При изменении кода — обновление происходит автоматически.
Для запуска требуется Node.js (версия 18+).
